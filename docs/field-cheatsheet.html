Field cheat sheet

INAV 9.0.1 · TBS Tango 2 · Baloo · Swordfish · Dolphin

Switch map

Switch↑ / unpressedmiddle↓ / pressed
SA left 2-pos DISARM — ARM
SB left 3-pos ANGLE ACRO MANUAL
SC right 3-pos no nav LOITER RTH
SD right 2-pos launch off — NAV LAUNCH
SE shoulder — — BEEPER (hold)
SF shoulder tap to toggle AUTOTUNE (sticky)

Pre-flight sequence

1
Power FC & radio. All switches ↑ / unpressed. No warnings expected.
2
Wait for GPS fix — check OSD sat count ≥ 8.
3
Verify OSD: battery voltage, video link, home-dir arrow present.
4
Launch with assist? Flip SD ↓ before arming.
5
Arm: SA ↓. (No GPS fix? Hold full-right yaw while flipping.)
6
Throttle up, throw.
7
Airborne: SD back to ↑ so you don't re-enter launch.

Routine flight

  • Default cruise → SB ↑ = ANGLE
  • Sportier → SB mid = ACRO
  • Glider / expert → SB ↓ = MANUAL
  • Goggles on/off → SC mid = LOITER
  • Find the plane → hold SE

Emergency

Lost orientation? SC ↓ = RTH. Plane climbs to 50 m, flies home.
Getting back control: walk SB back to ↑ before moving SC off ↓. RTH overrides flight-mode while active — exiting instantly hands over whatever SB is currently set to.

Mode priority (who wins)

  • RTH overrides everything except ARM.
  • LOITER overrides manual sticks for position, ANGLE / ACRO / MANUAL handle attitude underneath.
  • NAV LAUNCH is active only between arm and first successful climb-out; self-exits afterwards.
  • No flight-mode aux active = ACRO (implicit, via SB mid).

AUTO TUNE (Baloo / Dolphin only, once per plane)

  1. Climb to ≥ 80 m in ANGLE, calm air, plenty of open space.
  2. Tap SF once → latch on (listen for voice callout if configured). Plane will induce oscillations on pitch + roll.
  3. Fly figure-8s, let the controller sweep both axes for 30–60 s.
  4. Tap SF again → latch off. Values saved to the active PID profile.
  5. Land. Back at the bench, disable AUTOTUNE in the aux block so future mis-taps do nothing.
